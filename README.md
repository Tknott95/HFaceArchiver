# HFaceArchiver

#### INSTALLING
`pip install huggingface_hub`

INSTALL DOCS: https://huggingface.co/docs/huggingface_hub/en/installation

##### EXAMPLE
 **with default target**
`python script.py --repo <MODEL-NAME>`
 
 **with download loc**
`python script.py --repo <MODEL-NAME> --target <DOWNLOAD_LOC>`
